#### 安装教程

1.安装node

2.安装cnpm		

```
npm i cnpm -g
```

3.安装VUE命令行（脚手架）

```
cnpm i vue-cli -g
```

4.构建项目

```
vue init webpack myapp
```

5.布码云实现代码托管，并接通仓库。

(1)下载

```
git clone 您的仓库地址
```

(2)获取公钥

```
ssh-keygen -t rsa -C "您的邮箱"  
cat ~/.ssh/id_rsa.pub
ssh -T git@gitee.com
```

6.上传代码到仓库

```
（1）连接
git config --global user.name '用户名'
$ git config --global user.email '邮箱'
（2）上传
git add .
git commit -m 'init'
git push
```
